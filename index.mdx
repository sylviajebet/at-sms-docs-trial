# Send Thank You Messages Using Africa's Talking SMS API

This guide walks you through building a simple app to send thank-you messages to event attendees using the Africa's Talking Bulk SMS API.

✅ What You’ll Learn:
- Set up your Africa’s Talking account and API key
- Build a simple app (Python/Node.js) to send messages
- Deploy and test your app
